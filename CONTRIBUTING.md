# Contributing

¡Gracias por contribuir!

## Pull Requests

1. Fork el repositorio
2. Crea una rama: `git checkout -b fix/descripcion`
3. Haz cambios
4. Commitea y push
5. Abre PR

## Código

- Sigue las mejores prácticas del lenguaje
- Agrega documentación
- Mantén funciones pequeñas

## Testing

```bash
# Run tests
npm test  # or pytest -v
```
